## What is Rubber Numbers?
Rubber Numbers is an arbitrary integer size support library for C++.

### Features
- Scalable.
- Free.
- Open-source.
- Simple.
- Very readable.

### Platforms
- Cross-platform.

### Dependencies
- None, it just uses the standard C++ library.
